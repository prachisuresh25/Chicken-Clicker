Chicken Clicker
A serious platform for meaningless interaction.
Chicken Clicker is a useless website where the only thing you can do is click a chicken.
There is no goal. No reward. No progress.

The website tracks:
how many times you clicked
how much real time you wasted
how concerned the chicken becomes

If you stop clicking, the session ends.
